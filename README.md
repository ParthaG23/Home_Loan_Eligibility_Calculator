**Home Loan Eligibility Calculator** is a clean, interactive finance calculator running entirely in the browser. It helps potential home buyers estimate eligible loan amounts, monthly EMI repayments, and debt-to-income limits.

*Instant Projections · Interactive Sliders · Debt-to-Income Calculation · Zero Cookie Footprint*

</div>

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🧮 **Instant Eligibility Engine** | Computes maximum borrowing limit based on gross monthly income, existing EMIs, and age thresholds. |
| 🎚️ **Dynamic Range Sliders** | Drag inputs (Interest, Tenure, Salary) to watch calculation models update in real-time. |
| 📊 **Breakdown Cards** | Visual sections mapping Total Principal, Accrued Interest, and final EMI payouts. |
| 📱 **Responsive Grid Layout** | Adapts seamlessly to smartphones, tablets, and desktops. |

---

## 🧱 Tech Stack

* **Structure**: Semantic HTML5 markup
* **Styling**: Vanilla CSS3 (Custom properties, grid, flexbox layout)
* **Calculations**: Vanilla JavaScript (ES6+ Mathematical Formula Engine)

---

## ⚙️ How to Run Locally

No installations or local node servers are required. Everything runs directly in the client browser.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ParthaG23/Home_Loan_Eligibility_Calculator.git
   ```
2. **Launch the project**:
   - Double-click `index.html` to open the calculator instantly in your web browser.
